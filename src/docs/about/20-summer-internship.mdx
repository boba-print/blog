# 보바 하계 인턴 모집

세종대학교 재학생으로 이루어진 스타트업 기업 "**보바"** 에서 인턴 사원을 모집합니다. 

저희는 대학 캠퍼스 내의 **대기 시간,** **위생, 보안** 등의 문제로 짜증을 유발하던 무인 프린터기를 대체하는 차세대 클라우드 프린팅 서비스를 개발하고 있으며, 언제 어디서나 내 집처럼 프린터를 사용 할 수 있는 공유 프린팅 플랫폼으로 성장하는 것을 목표로 하고 있습니다.

현재 보바는 6명의 팀원으로 구성되어 있고 정부과제, 앤젤 투자 유치를 통해 4억원 규모의 Pre-A 단계의 자금 유치를 완료하였고, 90% 정도의 제품 개발 완료 상태로 세종대, 경희대, 인하대 등의 학교에서 제품 베타 테스트를 진행하고 있습니다.

저희는 9월 본격 제품 런칭에 맞춰 개발 마무리 작업을 함께 할 개발자 분을 찾고 있습니다. 단순 인턴 채용 뿐만 아니라 팀원으로 모집하여 같이 성장할 학우를 찾고 싶습니다! 실력과 업무량에 따라 추가수당 지급 가능합니다. 여러 포지션이 열려 있으니 많은 관심과 지원 바랍니다. 🥰

## 모집 부분

- **[Position 1 (FrontEnd)]** React.js, Redux.js 관련 유 경험자 **(우대, 급여 협의)**
- **[Position 2 (Backend)]** : Node.js(JavaScript) 또는 Firebase 관련 지식 보유자
- **[Position 3 (OS)]** : Windows OS 로컬 정책 설정 관련 지식 보유자.
- **[Position 4 (현장)]** : 현장 키오스크 설치 및 LAN 매설 **(급여 협의)**

## 채용 상세

### **Position1. Frontend 담당 업무 (채용 우대)**

보바의 Front End 개발자는 React.js 로 개발 된 Web App 들을 고객 피드백을 통해 취합된 User Story (고객 요구사항) 에 따라 개선하고 버그 안정화 작업을 합니다. 채용 우대 포지션으로 실력에 따라 화끈하게 추가 수당 지급.

- React.js, Redux.js 를 통한 Web App 유지 보수 및 개선작업
- Pupeteer.js 을 이용한 테스팅 자동화 작성

**자격 요건**

- React.js 와 Redux.js 를 이용한 프로젝트 경험이 있으신 분

**우대사항**

- Agile 개발 프로세스 경험자
- Firebase, Backendless 관련 지식 보유자
- 추후 스타트업 창업 희망자

### Position2. Backend 담당 업무

보바의 Backend 개발자는 Node.js, Firebase, Google Cloud 기반으로 개발 된 백엔드 소스의 성능을 개선하고, 테스팅 자동화를 위한 Pipeline 을 구성합니다.

- Firebase, Google Cloud 기반의 Backendless 서버 개발
- CI/CD 구성을 위한 Testing 코드 작성

**자격 요건**

- Node.js 를 이용한 프로젝트 경험 있으신 분

**우대사항**

- Firebase 이용한 프로젝트 경험 있으신 분
- 기타 Backendless 개발 경험
- 파일 관리 관련 프로젝트 유 경험자.
- 추후 스타트업 창업 희망자

### Position3. Windows OS 세팅 담당 업무

키오스크에 사용되는 Windows OS 를 설정하는 업무를 담당합니다. 로컬 정책, 레지스트리 설정 파일 제작 등의 업무를 담당합니다. (코딩 없음)

**자격요건**

- Windows OS 세부 설정에 관한 지식 보유자

**우대사항**

- 컴퓨터 부품과 조립에 흥미 있으신 분
- 구글링 및 영문 자료 해석 능통자
- 추후 스타트업 창업 희망자
- 강한 근성의 보유자

### Position4. 현장 키오스크 설치 업무

대학 현장에 키오스크 설치 업무를 합니다. LAN 설치 및 IP 세팅하고 프린터 등 무거운 짐 옮기는 일 많이 합니다. 코딩과 관련 없는 업무입니다. 수고한 만큼 추가수당을 지급하겠습니다.

**자격요건**

- 현장 업무 희망자

**우대사항**

- "3대 500" 준하는 언더아머 착용 가능 자.
- 2종 보통 이상의 면허 보유 및 자동차 운전 가능자.
- 추후 스타트업 창업 희망자
